Shows the official currency exchange rate from US Dollars to Costa Rica Colones.

== Description ==

Include in your site the official exchange reference rate for US Dollars to Costa Rica Colones.

The information is obtained directly from the Central Bank of Costa Rica web service.

== Installation ==

###1.Extract and upload the plugin###

Extract all files from the ZIP file, **keeping the file/folder structure intact**, and then upload the `xrcr-widget/` it to `/wp-content/plugins/`.

###2.Activate the plugin###

Go to the admin area of your WordPress install and click on the "Plugins" menu. Click on "Activate" for the "Costa Rica Currency Exchange Rate" plugin.

###3.Widget usage###

1. In the WordPress admin area go to "Appearance", "Widgets".
1. Drag and drop the widget "Costa Rica Currency Exchange Rate" to the sidebar on the right to activate it.

== Frequently Asked Questions ==

= I follow all the steps to install the plugin and activate the widget but the numbers aren't showing at all =

It's possible that there is a connection problem to the Central Bank of Costa Rica web service. Just try later.
